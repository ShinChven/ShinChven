# ShinChven

Full-stack engineer based in New Zealand. I build product-focused software across web, cloud, automation, and AI workflows, with most of my recent work centered on developer tools and practical AI applications.

## Current Focus

- Building AI-native products and internal tools
- Shipping across TypeScript, React, Node.js, .NET, Python, and cloud platforms
- Turning rough workflows into usable systems with clean DX

## Main Projects

### [remix-studio](https://github.com/ShinChven/remix-studio)

Self-hosted workspace for AI image generation projects. It combines a React frontend, Hono server, PostgreSQL via Prisma, and S3-compatible storage for managing prompts, assets, jobs, and exports.

### [pilot-banana-azure](https://github.com/ShinChven/pilot-banana-azure)

Automation platform for high-volume social media campaigns. Built on Azure Functions, Durable Functions, Cosmos DB, Blob Storage, and a React dashboard, with MCP support for AI assistants.

### [comfydl](https://github.com/ShinChven/comfydl)

Python CLI for downloading and organizing ComfyUI models from sources like Hugging Face and Civitai. Focused on practical model management, smart placement, auditability, and safe removal.

### [git-gpt](https://github.com/ShinChven/git-gpt)

Multi-provider AI CLI for git workflows. It helps generate commit messages, issues, changelogs, and quality checks using OpenAI, Azure OpenAI, Ollama, Claude, and Gemini-compatible setups.
